description: One-step session checkpoint for Worldbuilder.
---

Checkpoint the session so the window can be cleared losslessly:
1. Run the gate (`/gate`) and confirm green - or note honestly if it could not be run (no headless runner yet).
2. Update the newest handoff (or draft a new one), `STATUS.md`, the CodeMap, and Engineering Lessons as needed.
3. Commit the record (do not push unless asked; if pushing, confirm first).
4. Print a self-contained next-session prompt.

Only wrap at a clean stopping point (a coherent unit done + committed), never mid-change.
