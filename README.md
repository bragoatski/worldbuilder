# Worldbuilder

A procedural terrain and ecology simulator: a single self-contained HTML file (vanilla JS + a 2D canvas) that generates a world (volcanic terrain, climate, 16 biomes) and then runs a living ecosystem on top of it (flora, herbivores, carnivores with evolution).

## Run it
Open `index.html` in any modern browser. No build step, no dependencies (fonts load from Google Fonts when online).

## Controls
Start / Pause / Step / Reset in the toolbar. Click a tile to inspect it. Mouse wheel zooms, right-drag pans, double-click resets the view. Press `T` to run the in-page test suite.
